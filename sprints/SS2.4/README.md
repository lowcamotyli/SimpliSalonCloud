# SimpliSalonCloud — Release SS2.4

Branch: `SS2.4`
Base: `main` (post SS2.3)

## Overview

SS2.4 wprowadza rozbudowę UX zarządzania usługami (typy cen, tworzenie usług, galeria, dodatki),
nowe funkcje kalendarza (nieobecności, rezerwacje czasu), widok listy klientów, nowe raporty
oraz rozbudowę sprzętu. Zawiera też hotfixy zgłoszone przez klientkę.
