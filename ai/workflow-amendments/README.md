# Workflow Amendments

RFC-style notes for versioned changes to roles, enforcement, or handoff schema.

## When to write an amendment
- Change to worker roles (Claude / codex-main / codex-dad)
- Change to enforcement mechanism (rules, hooks, CI gates)
- Change to WORK.md schema
- Any decision that deviates from the architecture in `docs/architecture/deep-research_AgenticFramework.md`

...
